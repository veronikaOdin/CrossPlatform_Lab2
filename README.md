# Лабораторна робота №2
Дисципліна: "Кросплатформне програмування"
Виконала: Одинець Вероніка

## Опис проєкту
У проєкті продемонстровано три різні стратегії серіалізації об'єктів у Java на прикладі моделі навчального процесу:
* `task1_1/Version1.java` — стандартна серіалізація через інтерфейс `Serializable`.
* `task1_2/Version2.java` — ручне керування серіалізацією з використанням `transient` полів та шифруванням балів.
* `task1_3/Version3.java` — повне ручне керування через інтерфейс `Externalizable`.
